Multi-modal Mentor Quality Auditing System

Reviewing mentor session recordings manually was slow, inconsistent, and impossible to scale. Built a system that transcribes long-form sessions and evaluates them against a rubric using LLM-based scoring.

What I learned in production LLM rubric scores drifted across sessions when the prompt wasn't anchored with scored examples. Added few-shot calibration examples per rubric dimension.

ECCN Trade Compliance RAG Classifier

Classifying export-controlled items under ECCN codes required deep regulatory knowledge and was error-prone manually. Built a constrained reasoning RAG system using BGE embeddings, FAISS, and Gemini with re-ranking.

What I learned in production Semantic similarity alone wasn't enough for regulatory precision. Adding a re-ranking layer over retrieved candidates lifted Recall@5 by 21.5 points.
